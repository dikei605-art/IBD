Examples · 질문 예시

이런 걸 물어볼 수 있어요

모델명과 증상만 자연어로 입력하면 됩니다. ※ 아래 답변은 예시입니다.

설계변경 이력
380 정전기 개선은 언제부터야?
InBody 380 기준, 정전기 개선 설계변경은 2021년 11월 생산분(S/N 21110001~)부터 적용되었습니다.

변경 내용: 핸드 전극부 정전기 방지 코팅 추가 (MAR-2021-047).
해당 S/N 이전 제품은 현장 조치 필요합니다.
※ 예시 답변
CS 매뉴얼 · AS 이력
770에서 와이파이 모듈 교체했는데 클라우드가 안돼
InBody 770 Wi-Fi 모듈 교체 후 클라우드 미연결 시:

1. 장비 설정 → 네트워크 → SSID 재등록
2. 클라우드 서버 주소 초기화 후 재입력
3. 방화벽 포트(443) 오픈 여부 확인

유사 AS 이력: 모듈 교체 후 MAC 주소 변경으로 미연결 사례 다수.
※ 예시 답변
CS 매뉴얼 · 기준값
인체지그로 인바디770 측정 시 체지방률 기준
InBody 770 인체지그(RC Unit) 측정 기준값:

체지방률(BFP): ±1.5% 이내 (20회 연속 측정 평균 기준)
체수분: ±0.3L 이내

기준 초과 시: 전극 청소 → 지그 캘리브레이션 → 재측정 순서로 조치.
※ 예시 답변
파일을 열 필요 없이,
질문하면 바로 답변이 옵니다.
사내 네트워크 연결 시 누구나 웹 브라우저에서 접속 가능합니다.

Roadmap · 향후 계획

앞으로의 방향

단순한 검색 도구를 넘어, 현장 AS 기사의 실질적인 도구로 발전시킵니다.

Phase 1 · 현재
사내 PC 웹 챗봇
사내 네트워크 웹 브라우저에서 4개 DB 자연어 검색 및 AI 답변. 현재 운영 중.
✅ 운영 중
Phase 2 · 단기 검토
모바일 접근성 개선
현장에서도 접근 가능하도록 모바일 최적화 또는 앱 형태 제공 검토 중. 구현 방식 미정.
🔄 검토 중
Phase 3 · 최종 목표
현장 AS 완결 도구
사진·영상 포함 점검 가이드, 자재 사진 매칭 기능 추가.
비기너 AS 기사가 이것 하나로 AS를 완료할 수 있는 수준.
